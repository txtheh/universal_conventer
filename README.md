# Universal Converter

Конвертер неотрицательных рациональных чисел между системами счисления с основаниями от 2 до 500.

## Возможности

- Целые числа, конечные и периодические дроби.
- Длина входной строки до 100 000 символов.
- Цифры: `0-9`, `A-Z` (10–35), `[число]` (>35).
- Точные вычисления без `float`/`double` (собственная длинная арифметика).
- Обрезка результата при превышении 100 000 символов.
- Графический интерфейс (Qt).
- Загрузка из файла и сохранение результата.

## Формат ввода


- Точка разделяет целую и дробную части.
- Период в круглых скобках, может отсутствовать.
- Квадратные скобки `[n]` — одна цифра со значением `n`.
- Значение каждой цифры строго меньше основания `p`.

**Примеры:**  
`123`, `0.5`, `0.(3)`, `1.A(B)`, `[222][223]`, `0.[10][20]([30])`

**Ошибки:** пустая строка, недопустимые символы, пустой период, лишние символы после `)`, некорректные `[]`, значение цифры ≥ `p` и др.

## Формат вывода

- Цифры `0-9`, `A-Z` (10–35), `[n]` для `n ≥ 36`.
- Период в круглых скобках.
- `0` → `"0"`.
- При превышении `MAX_OUTPUT_LENGTH = 100000` символов вывод обрезается и добавляется сообщение.

## Алгоритм

1. Разбор входной строки → обыкновенная дробь `BigFraction`.
2. Сокращение дроби (через `gcd`).
3. Перевод целой части: деление на `q` → цифры в системе `q`.
4. Перевод дробной части: умножение остатка на `q`, выделение целой части (цифра), отслеживание остатков для поиска периода.
5. Сборка результата с учётом ограничения длины.

## Классы (C++17/20)

| Класс | Назначение |
|-------|-------------|
| `BigInteger` | Длинная арифметика (неотрицательные). Внутреннее основание `1e9`. Операции: `+, -, *, /, %, сравнения, gcd, Карацуба`. |
| `BigFraction` | Рациональное число (числитель/знаменатель). Сокращение, арифметика, целая часть. |
| `DigitTokenizer` | Разбор строки на цифры, точку, скобки. |
| `InputValidator` | Проверка корректности ввода. |
| `BaseParser` | Преобразование строки в `BigFraction` для системы `p`. |
| `BaseFormatter` | Преобразование `BigFraction` в строку системы `q`. |
| `BaseConverter` | Фасад конвертации. |
| `MainWindow` | GUI (Qt). |

## Примеры

| p  | q  | input         | output          |
|----|----|---------------|-----------------|
| 10 | 2  | 10            | 1010            |
| 10 | 2  | 0.5           | 0.1             |
| 10 | 2  | 0.1           | 0.0(0011)       |
| 10 | 16 | 0.1           | 0.1(9)          |
| 2  | 10 | 1.0(0)        | 1               |
| 16 | 2  | A.F           | 1010.1111       |
| 16 | 10 | 0.1(2)        | 0.0708(3)       |
| 100| 10 | 0.[10][20]    | 0.102           |
| 10 | 3  | 0.5           | 0.(1)           |
| 224| 10 | [222][223]    | 49951           |

## Ограничения

- `2 ≤ p, q ≤ 500`
- Максимальная длина вывода: `100000` символов.
- Запрещены сторонние библиотеки длинной арифметики и вещественные типы.
- Кроссплатформенность (Windows, Linux, macOS).
